# 📚 Online Book Store Website

This project is a **static multi-page website** designed for an online book store. It includes several web pages that simulate a basic e-commerce interface for browsing and purchasing books.

---

## 🚀 Project Overview

The website consists of five main pages:

1. **Home Page** – Introduction and navigation
2. **Login Page** – User login form
3. **Catalogue Page** – Displays available books
4. **Registration Page** – User signup form
5. **Cart Page** – Shows selected books and total price

---

## 🛠️ Technologies Used

- HTML5
- CSS3
